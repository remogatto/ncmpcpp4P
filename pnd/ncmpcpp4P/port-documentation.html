Setup your SD

Place your stuff as follows:

  • Your music collection goes in ncmpcpp4P.remogatto/music
  • Your playlists files go in ncmpcpp4P.remogatto/playlists

Override default configuration

  • Edit mpd.conf in ncmpcpp4P.remogatto/ to override default mpd server configuration
  • Edit terminal.opts file in ncmpcpp4P.remogatto/ to override default options passed to the terminal (in order to control its appearance)
  • Edit config and keys files in ncmpcpp4P.remogatto/.ncmpcpp to override default ncmpcpp options
